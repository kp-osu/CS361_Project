Hangman: Wordle Edition


Game Settings

Show Correct Letters

Of the words you've guessed, this will display the letters which are in the correct positions

Show Contained Letters

Of the words you've guessed, this will display the letters which are in the word somewhere

Prevent Invalid Words

Won't count invalid words (too short, too long, not a word) as a guess

Prevent Used Letters

Prevents you from attempting to use a letter for a second time that is known not to be in the word

Use Time Limit

Makes it so that you only have a set amount of time to guess each word
